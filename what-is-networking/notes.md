## What is Networking ?
Networking = communication between computers

## Simple View
You → Internet → Server → Response → You

That entire communication process is networking.


**While communicating with one device to another devices needes some ways and respect to talk with them.**

That respect are called <mark>Protocols</marks>.

There are many protocols to communicate between two devices.


## Because this networking is between computers So, we can say <mark>_**Computer Networking**_</mark>.
